# Game Server Configuration

## Environment Variables

```env
PORT=3001
NODE_ENV=production
MAX_PLAYERS=100
ADMIN_KEY=your-secret-admin-key-change-this
```

## Deployment

This server can be deployed to:
- Railway
- Render
- DigitalOcean App Platform
- Heroku
- Any Node.js hosting platform

## Usage

```bash
# Install dependencies
npm install

# Start server
npm start

# Development mode
npm run dev
```
