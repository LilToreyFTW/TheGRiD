About GRiD

GRiD is an immersive 3D video game featuring a massive world with 10,000 unique moped bikes, multiplayer functionality, and an advanced 3D printing system.

Features

  • 10,000 unique moped bikes with different body kits
  • Multiplayer server - everyone sees everyone
  • BORTtheBOT 3D printer system
  • Advanced graphics with ray tracing support
  • NVIDIA RTX GPU optimization
  • Mod support for custom content
